Фронтенд в Docker не включён — запускается отдельно через Vite. Полная последовательность:

**Терминал 1 — бэкенд:**
```bash
cd C:\Users\Mr_GooRoo\rca-analyzer
git pull origin main
docker-compose up -d
docker-compose exec api alembic upgrade head
```

**Терминал 2 — фронтенд:**
```bash
cd C:\Users\Mr_GooRoo\rca-analyzer\frontend
npm run dev
```

| Сервис | URL |
|---|---|
| Frontend | http://localhost:5173 |
| Swagger UI | http://localhost:8000/docs |

Пересборка образа нужна только если менялся `Dockerfile` или `pyproject.toml` — тогда замени первую команду на `docker-compose up -d --build`. После обычного `git pull` достаточно просто `up -d` (код монтируется через volume, `--reload` подхватывает изменения автоматически).
